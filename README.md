*capabilities*
-> It can perform basic arithmetic functions (+,-,/,*,%,sqrt). also adding decimal point'.'
-> Add a negative sign through '+/-' button.
-> You can view steps of calculations made throw the up/down arrow.
-> Backspace through the 'pause' type looking button in the bottom left.
-> Set custom tax rate through 'rate' button , add or subtract the set rate through 'GST+' and 'GST-' buttons.
-> Rest all like a normal calculator ,numbers ig :)
